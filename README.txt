by Isoheptane
Prototype of ESCTP(Even Simpler Chat Transmission Protocol)
Runtime Requirement:
  Server: .NET Core 3.1
  Tester: .NET Core 3.1
  Client: .NET Framework 3.0
You can also use tester as a client.
Client;
  Use "#" to indicate a client command.
  Use "/" to indicate a common server command.
  Use "!" to indicate a admin server command.
Tester:
  Use "#" to indicate a common server command.
  Use "!" to indicate a admin server command.
